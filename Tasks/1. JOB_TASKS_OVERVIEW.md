## üí∞ Free Tier Technologies Requirement

All tasks MUST use 100% free technologies and services (with generous free tiers):

### ‚úÖ Approved Free Services

- **Caching**: Upstash Redis (10K requests/day free)
- **Real-time**: Socket.io (self-hosted, free)
- **Video**: Daily.co (10K minutes/month free) OR self-hosted Jitsi
- **SMS**: Twilio (trial credits) OR Termii (free tier)
- **Push**: Web Push API (native, free)
- **Monitoring**: Sentry (5K errors/month free) + Winston (self-hosted)
- **Database**: PostgreSQL (current)
- **Search**: PostgreSQL Full-Text Search (free, built-in)
- **Email**: Nodemailer with Gmail (current setup)
- **File Storage**: Cloudinary (25GB free tier, current)
- **CI/CD**: GitHub Actions (2K minutes/month free)
- **Hosting**: Vercel (Frontend) + Render/Railway (Backend) free tiers

### ‚ùå Avoid Paid-Only Services

- AWS SES (paid)
- Twilio beyond trial credits
- Premium monitoring services
- Elasticsearch (use PostgreSQL full-text search instead)
